usageData

Anonymous usage data

When enabled, the system will submit anonymized usage data to a HiFiBerry server. This helps us to understand what features are used. Without these data it is not possible for us to spend our development efforts on the right features. We might even remove some features if we think that nobody is using them.

What's shared?

Anonymous usage about what features and sources are used, no data about the music you're listening.

You can have a look at these data by yourself by logging in and running 

/opt/hifiberry/bin/report-dump
.
